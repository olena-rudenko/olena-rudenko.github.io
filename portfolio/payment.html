The purpose of this work is to develop a convenient and easy to use payment form.


Description:

The design developed for all possible errors, it is easy to use. Design can be adapted to any type of sales. The kit contains forms for various types of payment method.




Typing


The form is designed in the way that data entry takes place very quickly with considering peculiarities of usability


Sending


If there is a slight delay during the sending of data, the user is informed that the process is in progress.


Errors


The design provides for all possible errors that may arise during the use of the form.
